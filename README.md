**Smart FJWU Campus Navigator Web**

A modern web-based campus navigation system developed for students and visitors to explore university locations easily. The project integrates QR-based multimedia access and a feedback system using Firebase.

**Features**
Campus location navigation
QR code multimedia access
Student feedback system
Responsive UI design
Real-time data management
**Technologies Used**
HTML
CSS
JavaScript
React.js
**Future Improvements**
AI-based navigation
Indoor maps
Admin dashboard
Live notifications
